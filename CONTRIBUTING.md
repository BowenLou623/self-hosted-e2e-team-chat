# Contributing

Thanks for helping improve Team Chat Launcher.

Good contributions include bug fixes, documentation improvements, self-hosting safety improvements, privacy-preserving defaults, and conservative security clarifications.

## Issues

When opening an issue, include:

- What you expected.
- What happened.
- Operating system and Python version.
- Minimal reproduction steps.
- Redacted logs only when needed.

Do not upload real chat records, personal information, private keys, tokens, local databases, or runtime profiles.

## Pull Requests

- Keep changes focused.
- Add manual verification notes when behavior changes.
- Do not submit secrets or real user data.
- Do not add payment, subscription, public relay, hosted service, regulatory evasion, law-enforcement evasion, attack, or abuse-oriented features.
- Do not claim absolute security, absolute anonymity, or untraceability.

Contributors should follow lawful, responsible, and security-first principles.
